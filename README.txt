Gobit AI WORK - Demo
====================
This is a lightweight single-page application (no build tools) that simulates:
- Uploading a resume (.pdf/.doc/.docx)
- An "AI is analyzing" animation (spinner, dots, progress bar)
- Showing a simulated list of recommended companies

How to use:
1. Unzip the package
2. Open index.html in your browser
3. Click "AI WORK" in the top nav, upload a file, and press "เริ่มใช้งาน AI WORK"

Notes:
- This is a front-end demo only. No real AI or backend is included.
- You can improve by hooking a backend API to receive the uploaded file and return real recommendations.
